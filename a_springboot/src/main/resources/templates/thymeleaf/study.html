1. 변수 선언하고 출력하기
		변수 선언 :  th:with
		변수 출력 : ${...}
		
2. *{...} 선택 변수 표현식 
		th:object에 저장된 객체의 속성에 접근할 때 선택 변수 표현식을 사용해 속성값 만으로 해당 값을 사용할 수 있다.
		th:utext : escape 처리를 하지 않고 해당 요소의 텍스트 값을 지정한다. 
		
3. @{...} 링크 표현식
		url뒤에 괄호를 작성해 파라미터를 추가할 수 있다. ex) @{https://www.naver.com(test=100,name=hmd)}
